From the Volunteers menu, clicking on Volunteer Interest Form yields:

This form is used to allow an existing Contact in CiviCRM to become added to the list of available Volunteers for a mailing list. First Name, Last Name, and Email are required fields. Providing a Phone Number is optional. If a contact is not found matching the required fields, a new contact is created in CiviCRM.
